HTML Syntax & Nesting

An HTML element typically has an opening tag, content, and a closing tag. Some elements are empty (no content), like <img> or <br>.

Correct vs Incorrect nesting

Correct: <h3>Share Your <strong>Travels</strong></h3>

Incorrect: <h3>Share Your <strong>Travels</h3></strong>

Nesting rule: tags must close in the reverse order they opened.

Parent / child idea (DOM tree)

<article>
  <h4>Title</h4>
  <p>This is some <strong>text</strong>.</p>
</article>

In that example: <article> is the parent of the <h4> and <p> children.

Character entities are used for reserved characters or symbols, e.g. &lt; renders as <, &copy; renders as ©, and &trade; renders as ™.

<!-- Child directory -->

<!-- Parent directory -->
<a href="../about.html">About</a>

<!-- Root reference (site root) -->
<a href="/images/background.gif">Background</a>
